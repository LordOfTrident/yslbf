## Introduction
A broken and very slow brainf interpreter written in [YSL](https://github.com/yeti0904/ysl) that can
only run a simple hello world program.

## Quickstart
```sh
$ ysl main.ysl
Loaded program
Enter the path of the file to run
> examples/helloworld.bf
```
